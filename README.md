# ReScript Calculator

```zsh
npm install
```

## Dev

```zsh
npm run re:dev
```

```zsh
npm run dev
```

> Run two scripts sequentially at the same time. You can use tools like [concurrently](https://www.npmjs.com/package/concurrently), [npm-run-all](https://www.npmjs.com/package/npm-run-all), or just with two shells.

## Build

```zsh
npm run build
```
